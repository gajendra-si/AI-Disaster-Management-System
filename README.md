
# 🌍 AI Disaster Management System

An AI-powered Disaster Management Dashboard developed using **React.js, FastAPI, SQLite, Machine Learning, OpenWeather API, and React Leaflet**.

## 🚀 Features

* 🤖 AI Risk Analysis & Severity Prediction
* 🔥 Fire Alerts Dashboard
* 🌊 Flood Alerts Dashboard
* 📍 Current User Location Detection
* 🏙 Current City Name Detection
* 🌤 Real-Time Weather using OpenWeather API
* 🕒 Live Date & Time
* 🗺️ Live Disaster Map (OpenStreetMap + Leaflet)
* 🔴🟠🟢 Severity Colored Markers
* 📊 Disaster Analytics Dashboard
* 📈 Bar Chart Analytics
* 🥧 Severity Distribution Pie Chart
* 📥 PDF Report Download
* 🚨 Emergency SOS Alert
* 📞 Emergency Number (112)
* ☎️ Disaster Helpline (1078)
* 🗑️ Delete Reports
* 📱 Responsive & Modern UI

## 🛠 Technologies Used
